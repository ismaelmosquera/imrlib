Arithmetic Polynomial Example

This example demonstrates the basic arithmetic operations:
- Addition.
- Substraction.
- Multiplication.
- Division.

All of them are available for any number set.
- Naturals/Integers ( notice that a natural number is just an integer one greater than 0 ).
- Rationals.
- Reals.
- Complex.

In the case of division, there are classes to manage real, rational and complex.
If you wish to divide two integer polynomials, just convert to floating-point the array associated to the concrete polynomial.
In this example, we show some operations for several number sets, but not all,
since the example could grow arbitrary huge.
But you can be confident that all works properly.

To build an run the command line application example:

Maybe the example is already build. Anyway, if you want to rebuild it do as follows:

To build the example just type:

build

if you are using a MsWindows command line shell.
If you are under a Linux like shell, type:

./build.sh

To run the example type:

run

or

./run.sh

depending in what kind of shell you are.
