Cyclotomic Polynomial Example

This example from the command line will show a Menu as follows:

1: Show Menu.
2: Compute Pn(x)
3: Show coefficient for a given exponent.
4: Exit.

Option: 2 <Intro> 
Type an Integer value > 0
n: 9 <Intro> 
Print raw:
P[9] = [1, 0, 0, 1, 0, 0, 1]
Print formatted:
P[9] = [x^6 + x^3 + 1]

Option: 3 <Intro> 

Type an integer value for the required exponent [0 .. 6]
Exponent: 3 <Intro> 
Coefficient = 1

Option: 4 <Intro> 

bye.


The above is just an execution example.

To build an run the command line application:

Maybe the example is already build. Anyway, if you want to rebuild it do as follows:

To build the example just type:

build

if you are using a MsWindows command line shell.
If you are under a Linux like shell, type:

./build.sh

To run the example type:

run

or

./run.sh

depending in what kind of shell you are.
