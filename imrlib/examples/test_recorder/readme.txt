To build the example just type:

build

if you are using a MsWindows command line shell.
If you are under a Linux like shell, type:

./build.sh

To run the example type:

run

or

./run.sh

depending in what kind of shell you are.

The example is a command line application,
it looks like as follows:

1: show menu.
2: rec.
3: play.
4: stop.
5: save.
6: exit.

option: 

As you can see, you can record some audio using a microphone, play your recorded audio,
and save it to an audio file with the 
following extensions allowed:

- *.wav
- *.aif
- *.au

You can load and play your saved files using the iPlayer
in the examples folder or with any
comercial player.
